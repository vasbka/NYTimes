#Описание

Плагин предназначен для вывода ПОСЛЕДНЕЙ новости из газеты NY Times на заданную тематику.

#Инструкция по установке

В папке с плагинами WordPress создать собственную папку. Назвать как угодно.
Залить файлы в только что созданную папку. Зайти в админку WordPress и активировать появившийся плагин.
После можно добавлять шорт коды к вашим записям 
`[NYTimes cat="CATEGORY" period="PERIOD" counter="COUNT"]`
CATEGORY - категория которая вас интересует.
PERIOD - период за который вы хотите получить последние новости.
COUNTER - количество новостей по заданным параметрам.
Возможное значение PERIOD :
- 1
- 7
- 30

Возможные категории : 
- Arts
- Blogs
- Books
- Education
- Food
- Health
- Magazine
- Movies
- Multimedia
- Science
- Sports
- Style
- Technology
